To install Beaker packages (and dependencies) you'll need to get the repo file then copy it to your /etc/yum.repos.d/ directory.
There are two main repos. One containing packages needed for installing the Beaker server and required components, another for packages needed to run the Beaker client. Basic tasks needed by Beaker can also be found here. You can find the repo directory here. Please download the repo file that suits your requirements.
