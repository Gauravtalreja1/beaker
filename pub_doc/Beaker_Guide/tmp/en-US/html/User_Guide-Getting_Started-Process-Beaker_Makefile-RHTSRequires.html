3.2.1.2.15. RHTSRequires
RHTSRequires one or more. This field indicates the other beaker tests that are required to be installed on the test machine for the test to work.
This field can occur multiple times within the metadata. Each value should be a space-separated list of Beaker Test RPM name without the version or .rpm. Each Test must exist on the Beaker Scheduler.
For example: 
@echo "RHTSRequires:     rh-tests-distribution-hts-common" >> $(METADATA)
